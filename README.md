# Quiz Management System (Python + MySQL)

##  Description
This project is a console-based quiz application developed using Python and MySQL. It allows users to attempt quizzes across different difficulty levels (Easy, Medium, Hard) and provides an admin interface to manage quiz questions.

##  Features
- Multiple difficulty levels (Easy, Medium, Hard)
- Admin panel to add and remove questions
- Interactive quiz gameplay
- Score calculation system
- Topic-based quiz selection
- MySQL database connectivity

##  Tech Stack
- Python
- MySQL
- mysql-connector-python

##  How to Run

1. Install Python and MySQL
2. Install required dependency:
   pip install mysql-connector-python

3. Import database:
   Open MySQL and run:
   source database.sql;

4. Run the program:
   python main.py

## Project Structure
- main.py → Main program file
- database.sql → Database setup
- requirements.txt → Required dependencies
- project_report.pdf → Project documentation

## Learning Outcomes
- Database connectivity using Python
- SQL operations (INSERT, DELETE, SELECT)
- Menu-driven program design
- Basic backend logic implementation
